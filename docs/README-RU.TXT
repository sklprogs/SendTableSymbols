SendTableSymbols - это программа, позволяющая вставлять спец. символы (такие как àáâæßç) во внешнее приложение. Запустите программу (map.exe), щелкните на значке в трее, выберите спец. символ - и он будет вставлен в ту программу, которая у вас открыта в данный момент. Для выхода щелкните правой кнопкой на значке в трее.

Чтобы добавить дополнительные символы, отредактируйте map.cfg. Для его правки используйте специализированные редакторы, такие как AkelPad (akelpad.sourceforge.net), при сохранении уберите пометку с BOM, иначе, программа не сможет прочитать конфигурационный файл.

В данный момент поддерживается только Windows (ввиду таких компонентов как SendKeys и winico). Для запуска в виде скрипта, скопируйте winico0.6 в каталог tcl, например, C:\Python34\tcl\winico0.6\.

Приятной работы :-)
